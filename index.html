¿Qué es HTML y cuál es su función principal en el desarrollo web?

HTML es la ayuda y el encargado para hacer la estructra de lás páginas web, la cual define los elementos que esta contendrá, como por ejemplo:Títulos, párrafos, imágenes.

¿Qué son las etiquetas HTML y cómo se utlizan?

Las etiquetas son como palabras reservadas para definir una infinidad de elementos dentro de las páginas web,se encuentran dentro de los signos de 'mayor o menor' (<>). Ejemplos: a, img, div.

¿Qué es CSS y cómo se utiliza para dar estilo a una página web?

CSS se encarga de darle vida o estilizar una página web, la decora en otras palabras; funciona de tal manera que manda a llamar a las etiquetas o grupo de etiquetas para darles un estilo y poder definir desde tipo de letra, hasta como se posicionará un elemento.

¿Cuáles son las propiedades más importantes de CSS?

font-family, font-size,color,width,height,padding,margin,etc.

Explicación modelo de caja

Es un contenedor que contiene muchas propiedades, entre ellas están, bordes, márgenes, relleno y el contenido que se le asigne.

¿Qué es el diseño responsivo?

El diseño responsivo surge de la necesidad de presentar el contenido de una página web para muchas plataformas, y que en cada una de estas se visualice correctamente. La manera en la que funciona es por dimesiones, estas dimensiones son standares, ya que son medidas las cuales la mayoría de aparatos poseen.

¿Qué es JavaScript y como se intrega en el desarrollo de páginas web?

JS Es un lenguaje de programación para páginas web, la cual nos permite la codificación de elementos, como por ejemplo: enviar correos eletrónicos, escribir HTML por medio de botones, etc.

Document Object Model - ¿Cómo interactúa con HTML y CSS?

El DOM es una representación de la estructura de una página web, JS puede acceder a los elemntos que esta contenga y manipularlos, incluyendo el HTML (sus etiquetas), manipular CSS (Los estilos).

¿Cómo puede hacer JS una página web interactiva?

Esto será posible por medio de las actualizaciones en tiempo real, animaciones y elementos interactivos.

¿Qué es GIT y porqué es importante en el desarrollo web?

Git es una herramienta que facilita el trabajo colaborativo, para la creación de proyectos en equipo. Es importante ya que fomenta el orden en el desarrollo de proyectos, y tiene muchas herramientas para manipular los proyectos, su progreso.

Control de versiones con GIT

El control de versiones se lleva cada vez que nosotros guardamos nuestro trabajo, así vamos actualizando nuestro proyecto, y si deseamos podes deshacer algún movimiento; es decir, regresar a versiones pasadas.

¿Qué es GitHub y cómo facilita el desarrollo de proyectos web?

GitHub está asociado con git ya que se maneja de manera por código o interfaz gráfica, en cambio esta vez el proyecto se almacena en la nube, y podemos tener tambien un control del proyecto de manera gráfica. También podemos agregar colaboradores y hacer nuestros repositorios los cuales son proyectos, y estos nos traen infinidad de funciones.
